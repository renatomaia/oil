   luarocks install oil

To make the OiL packages available for any Lua application you may simply execute the application in the folder you unpacked OiL. Therefore, to give it a try, just execute the Lua stand-alone interpreter in the directory you unpacked OiL. However, to properly install OiL in your environment you must unpack it in the Lua package directory, which is defined by the luaconf.h file during the compilation of Lua. The default values for this directory are listed below:

Unix
Lua 5.1: /usr/local/share/lua/5.1/
Lua 5.2: /usr/local/share/lua/5.2/
Windows
Directory where the application (e.g. stand-alone interpreter) is installed. Therefore, to use OiL in Windows you may simply unpack the loop package in the same folder the Lua stand-alone interpreter (or other Lua application) is installed.

You can also define the directories Lua searches for packages by setting the <%=link("LuaManual.LuaPath", "LUA_PATH")%>.
