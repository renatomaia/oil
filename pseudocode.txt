ClientBroker:
	- newproxy(referencia para objeto, interface)
  - decodifica referencia para objeto
  - passa referencia decodificada para Proxy e nome da interface, se houver
ProxyFactory/TypedProxyFactory
	- se houver interface, tenta obter a interface no repositorio de interfaces
  - a referencia decodificada deve estar pronta para ser passada ao protocolo, quando for feita uma chamada a um metodo qualquer
	- guarda a referencia decodificada no proxy
	- retorna proxy criado

ServerBroker:
	- Cria um acceptor
	- Para criar o acceptor, e necessario configuracoes de ip e porta, criando um canal, usando o PassiveChannelFactory.
	- Passa o acceptor para a criacao de um dispatcher
	- Passa para o Dispatcher o novo acceptor
