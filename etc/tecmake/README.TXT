1. Go to an empty directory.
2. Run script 'etc/makepack'
3. Copy 'oil-0.5-beta.tar.gz' and 'oil-tecmake.tar.gz' to an empty directory in Tecgraf.
4. Go to that directory and unpack 'oil-0.5-beta.tar.gz'.
5. Get inside directory 'src' of the unpacked directory.
6. From there, unpack 'oil-tecmake.tar.gz'.
7. Log to a Windows machine in Tecgraf (pero).
8. Get inside directory 'src' of the unpacked directory.
9. Run script 'makewin.bat' twice.
10. Log out windows and go back to Linux.
11. Go to the root of the unpacked directory.
12. Run script 'src/packwin'.
13. Get all the resulting '*.zip' files and that's all.
